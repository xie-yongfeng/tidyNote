关于小记袋

小记袋是一款轻量便签收纳工具，支持剪贴板快速收纳、AI 一键标题、智能分区推荐和自定义 AI API，帮助你整理灵感、待办、摘录和碎片信息。

常见问题

  • 复制文字后打开 App，可使用剪贴板快速收纳功能。
  • AI 标题生成和智能分区推荐需要网络连接。
  • 如需使用自己的模型服务，可在 App 首页右上角 API 设置中配置兼容 OpenAI Chat Completions 格式的接口。
  • 如果 AI 功能不可用，请检查网络、API 地址、API Key 和模型名称是否正确。
